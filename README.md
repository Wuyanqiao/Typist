# Typist

一个 Windows 桌面打字助手。用户在草稿区输入内容，选择目标输入位置，设置输入速度和启动倒计时后，程序会自动点击目标位置，并把草稿内容逐字符发送到输入框、网页文本框或其他可输入控件。

## 运行

```powershell
dotnet run --project .\Typist\Typist.csproj
```

也可以直接运行构建后的程序：

```powershell
.\Typist\bin\Debug\net9.0-windows\Typist.exe
```

## 使用

1. 在草稿区输入要发送的内容。
2. 点击“选择目标”，窗口最小化后，在 3 秒内把鼠标移动到目标输入框上。
3. 程序会记录当前鼠标屏幕坐标，并在界面显示目标位置。
4. 设置“每字符间隔(ms)”和“启动倒计时(秒)”。
5. 点击“开始”，程序会在倒计时结束后自动点击目标位置，再开始输入。
6. 输入过程中可点击“停止”，或使用停止热键。

如果没有选择目标，程序仍会沿用旧模式：倒计时结束后输入到当前焦点位置。

全局热键：

- `Ctrl+Alt+S`：开始输入
- `Ctrl+Alt+Q`：停止输入

如果默认热键被占用，程序会自动尝试备用热键：

- `Ctrl+Alt+F11`：开始输入
- `Ctrl+Alt+F12`：停止输入

## 注意

- 程序使用 Windows 鼠标点击和 `SendInput` 向目标位置发送键盘输入。
- 目标位置按屏幕坐标保存；如果目标窗口移动、页面滚动或输入框位置改变，请重新选择目标。
- 如果目标程序以管理员权限运行，本程序也需要以管理员权限运行，否则 Windows 可能会阻止输入。
- “换行使用 Enter”适合网页文本框和大部分多行输入框；如果目标控件对 Enter 有提交行为，请先关闭该选项。
